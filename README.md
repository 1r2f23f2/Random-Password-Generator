# Random Password Generator (Генератор случайных паролей)
**Автор:** Камила Мухаметзянова
## Описание
Простое настольное приложение для генерации случайных паролей с возможностью выбора типов символов (цифры, буквы, спецсимволы), настройки длины пароля (от 4 до 32) и сохранения истории паролей в JSON-файл.
Приложение написано на Python с использованием библиотеки `tkinter` для GUI и `random` для генерации. История автоматически подгружается при запуске.
## Примеры использования
Запустите приложение:
```bash
python main.py
